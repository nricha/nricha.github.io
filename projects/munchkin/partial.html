Munchkin - Kids Recipe App

Goal: To design a recipe mobile app that is user-centred, by focussing on both, UX and UI design processes. For UX, the process involved competitive analysis and user research, creating user personas, defining MVP and user stories, designing a user flow, creating paper prototypes and usability testing. The UI design process included creating mood boards, wireframing, running preference tests, building a responsive design, mockups, creating a style guide and user testing.
