Flower Illustration

Goal: To create a realistic flower illustration by using various tools in Adobe Illustrator like pencil, pen, mesh, pathfinder, brushes and shapes. The process included creating outlines using pen or pencil tool and then using live paint to give it a base color. After that we had to create a gradient mesh to be able to extract the right colors. Finally, the details were to be added by using different profiles, brushes and effects.
